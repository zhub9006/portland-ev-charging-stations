# portland-ev-charging-stations

Portland downtown EV charging station data for **green transportation mapping** — open data contribution for sustainable commuting from downtown Portland to Beaverton.

> 📍 **Downtown Portland** (45.5159, -122.6822) → **Beaverton** (45.4872, -122.8038) | ~12.8 km via Sunset Highway

## Quick Stats

- **11 stations** surveyed within 5 km of downtown Portland
- **3 DC Fast** chargers (1 Tesla Supercharger @ 250 kW, 2 multi-protocol)
- **2 Multi-protocol** stations (J-1772 + CCS + CHAdeMO — works with virtually every EV)
- **Best L2 coverage**: Blink + ChargePoint dominate the 500–700 m zone near downtown core

## 🤝 Contributing

This is an open data project. To contribute:
1. Submit issues to report new stations or corrections
2. Fork and submit PRs with updated station data
3. Share via Open Charge Map ([openchargemap.org](https://openchargemap.org)) — global registry of EV charging stations (144 stars, MIT license)
